# 一致するまで終われま10

Zoom向けの「イッセーので」系マルチプレイゲーム。全員で「せーの！」で同じ答えを狙い、**累計10回一致**したらクリア（帰れる）。

各自の端末（スマホ/PC）のブラウザから参加し、ホストが目視で一致を判定します。Zoomは画面共有＆通話用に使い、実際の操作はこのアプリで行うイメージです。

## 構成

- `server/` — Node.js + Express + Socket.IO（ルーム管理・状態同期）
- `client/` — Vite + React（ホスト/プレイヤー画面）

## セットアップ

```bash
npm run install:all   # ルート + client の依存をインストール
```

## 開発（ローカル）

```bash
npm run dev
```

- サーバー: http://localhost:3001
- クライアント(Vite): http://localhost:5173 ← ここを開く

同じLAN内の別端末からは `http://<PCのIP>:5173` でアクセスできます（Viteが `host: true` 設定済み）。

## 本番ビルド & 起動

```bash
npm run build   # client をビルド
npm start       # サーバーが client/dist を配信（http://localhost:3001）
```

## 遊び方

1. ホストがニックネームを入れて「ルーム作成」→ 4文字のルームコードが出る
2. 参加者はコードとニックネームで参加
3. ホストがお題を入力してスタート
4. 全員が「せーの！」で回答を入力しロック（全員そろうと自動オープン）
5. ホストが目視で一致数を判定して加算（表記ゆれは柔軟にOK）
6. 累計10回一致でクリア 🎊

## カスタマイズ

- 目標回数: `server/index.js` の `GOAL`
- 最大人数: `server/index.js` の joinRoom 内（現在12人）
